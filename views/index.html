Simply Welcome

A Go project used as a starting point for web applications. No need to look how to structure the application on every new project, clone simple and start working.

Features & Third party packages

  • Beego for most of things
  • Gorilla websocket for websockets (currently server to clients only)
  • Configuration files for Nginx, SystemD and Supervisord
  • Starting websocket client models (rooms-clients)

You can also check http://localhost:8080/api/v1/websocket a small API route starting point

How to use

Clone, change name at the following files:

  • Project directory name
  • Project name at conf/app.conf
  • gosimple.com.nginx to your project name/domain (needs to be done inside the config file too)
  • gosimple-supervisor.conf to your project name/domain (needs to be done inside the config file too)
  • gosimple.service to your project name/domain (needs to be done inside the config file too)

By doing the above, you have an app ready to run at server boot, Nginx vhost and SSL ready.
